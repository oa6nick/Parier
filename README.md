# Pariall - P2P платформа пользовательских пари

Современный фронтенд для платформы Pariall, построенный на Next.js 15 с TypeScript и Tailwind CSS.

## Описание

Pariall — это социальная платформа, где пользователи создают и принимают пари друг с другом на реальные события (спорт, экономика, политика, технологии, развлечения). Пользователь не играет против платформы — он публично формулирует прогноз и подтверждает уверенность деньгами и репутацией.

## Технологии

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** для стилизации
- **Lucide React** для иконок
- **date-fns** для работы с датами

## Структура проекта


## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build

# Запуск production сервера
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Основные страницы

### 1. Лента пари (`/`)
Вертикальная лента карточек пари с фильтрами по категориям. Каждая карточка содержит:
- Информацию об авторе (аватар, рейтинг, процент побед)
- Заголовок и описание пари
- Сумму пари, коэффициент, потенциальный выигрыш
- Статус и дедлайн
- Кнопку "Ответить" для принятия пари
- Социальные элементы (комментарии, количество пари)

### 2. Создание пари (`/create`)
Форма создания пари с двумя способами ввода:
- Текстовый ввод
- Голосовой ввод с ИИ (UI готов, функционал на моках)

Обязательные поля:
- Заголовок
- Краткое и подробное описание
- Формулировка исхода
- Категория
- Сумма и коэффициент
- Дедлайн и дата события
- Источник проверки результата

### 3. Профиль (`/profile`)
Страница профиля пользователя с:
- Информацией о пользователе (аватар, рейтинг, процент побед)
- Статистикой (дата регистрации, локация, общее количество пари)
- Тегами интересов
- Карточками статистики (активные пари, выиграно, общий выигрыш, рейтинг)
- Табами для фильтрации пари (Все, Активные, Завершённые)
- Списком пари пользователя

### 4. Рейтинг (`/rating`)
Страница рейтинга игроков с:
- Статистическими карточками (всего игроков, всего пари, общий призовой фонд)
- Фильтрами сортировки (Топ по заработку, Самые активные)
- Списком игроков с медалями ранга (#1 золото, #2 серебро, #3 бронза)
- Информацией о каждом игроке (аватар, имя, рейтинг, процент побед, заработанная сумма)
